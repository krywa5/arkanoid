Name Type Attributes Default Description
spriteX number The x-axis coordinate of the top left corner of the sub-rectangle of the source image to draw into the destination context.
spriteY number The y-axis coordinate of the top left corner of the sub-rectangle of the source image to draw into the destination context.
width number The width of the sub-rectangle of the source image to draw into the destination context
height number The height of the sub-rectangle of the source image to draw into the destination context.
spritesImage CanvasImageSource An element to draw into the context.
x number X position of sprite in canvas
y number Y position of sprite in canvas
numberOfSprites number <optional>
 1 Default = 1, Number of sprite images in the main image
offset Offset <optional>
 {x: 0, y: 0} Default = {x: 0, y: 0}, Offset to draw
